# lemon-heart-心理网站Vue项目

> 本项目采用Vue-CLI+SpringBoot进行前后端分离，为有需求的人提供心理文章阅读、问题发布，心理测试……主要功能包括问题发布、创意讨论、文章回答点赞功能
>
> 前端采用Vue2+Vue-router+elementUI
